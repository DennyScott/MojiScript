# JoelScript net/axios

JoelScript net/axios example

## Install

```bash
# clone repository
git clone https://github.com/joelnet/JoelScript.git

# enter directory
cd JoelScript/examples/net-axios

# install dependencies
npm ci
```

## Run

```bash
npm start
```

You should see the output

```

- Luke Skywalker (male)
- Anakin Skywalker (male)
```
