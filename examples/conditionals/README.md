# MojiScript IfElse Simple

MojiScript IfElse Simple example.

## Install

```bash
# clone repository
git clone https://github.com/joelnet/MojiScript.git

# enter directory
cd MojiScript/examples/ifElse-simple

# install dependencies
npm ci
```

## Run

```bash
# run if/else
npm run ifelse

# run conditional 1
npm run cond

# run conditional 2
npm run cond2
```
